# Address Book

A simple web-based tool to manage and organize your contact list.

## Link

- URL Deployment: https://addressbooksya.vercel.app/
- Repository : https://github.com/lesyaasbll/addressbook

## Main Features

- Add new contacts
- Edit and remove existing contacts
- Search contacts by name or number or email

## Tech Tack

- HTML – For structure
- CSS – For styling
- JavaScript – For interactivity
